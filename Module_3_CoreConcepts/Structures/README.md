# Module 3: Structures Practice

- [ ] **q004:** Define a structure `Point` with `x` and `y` coordinates. Write a function to calculate the Euclidean distance between two given points.
- [ ] **q010:** Define a structure `Book` (title, author, price). Write a program to dynamically allocate memory for `n` books and sort them by price.
- [ ] **q015:** Create a nested structure containing an `Employee` (name, ID) and their `DateOfBirth` (day, month, year), and write a function to print it.
- [ ] **q019:** Write a function that accepts an array of structures representing students and returns the average marks of the class.
- [ ] **q023:** Define a `Rectangle` structure (length, width) and write a function to return its area, passing the structure by reference.
- [ ] **q031:** Define a `Date` structure. Write a function that takes two dates and returns the number of days between them.
- [ ] **q035:** Define a `Car` struct (model, year, price). Create an array of 3 cars, pass it to a function, and print the details of the newest car.
- [ ] **q040:** Write a C program to sort an array of structures based on a specific string field using the standard library `qsort()` function.
- [ ] **q045:** Define a structure `Fraction` with numerator and denominator. Write a function to multiply two fractions and return the simplified result.
- [ ] **q050:** Create an array of structures for a `Library` system. Write a function to search for a book by its title and return a pointer to the struct.
- [ ] **q056:** Define a `Node` structure for a singly linked list (data, next pointer). Write a function to dynamically create a node, initialize it, and return its pointer.
- [ ] **q063:** Define a `Time` structure (hours, minutes, seconds). Write a function to add two time periods and properly carry over seconds and minutes.
- [ ] **q069:** Create a structure `Item` with a union inside it to hold either an integer ID or a string ID, along with a type flag to track which is currently active.
- [ ] **q082:** Define a structure `Flight` and write a function that takes an array of flights and sorts them chronologically by departure time.
- [ ] **q086:** Write a program to simulate a stack using an array inside a structure, implementing `push` and `pop` functions.
- [ ] **q090:** Define a structure `Team` (name, points). Create an array of pointers to `Team` structures and sort the pointers based on points to maintain original struct memory locations.
- [ ] **q095:** Define a `Rectangle` structure and a `Circle` structure. Use a `void*` pointer to write a generic function that can print the area of either, based on a type identifier passed alongside it.
- [ ] **q099:** Create nested structures `Address` and `Person`. Write a function that performs a deep copy of one dynamically allocated `Person` (and their `Address`) to a newly allocated `Person`.
