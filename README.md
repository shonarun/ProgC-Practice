# ProgC-Practice

A dedicated repository for tracking coursework, laboratory experiments, and problem-solving for **CS1L002: Programming in C**. 

This repository is organized strictly by syllabus modules to maintain a clean file tree and ensure rapid iteration through problems. All solutions prioritize low-overhead execution and bare-metal fundamentals.

---

## 📂 Repository Structure

The workspace is divided into core conceptual modules and formal lab experiments:

* **`Module_1_Intro/`**: Data types, scope, and I/O statements.
* **`Module_2_ControlFlow/`**: Branching and iteration.
* **`Module_3_CoreConcepts/`**: Arrays, functions, pointers, and dynamic memory allocation.
* **`Module_4_Search_Sort/`**: Linear/binary search and fundamental sorting algorithms.
* **`Module_5_FileHandling/`**: File I/O operations.
* **`Labs/`**: Formal laboratory experiment deliverables.
